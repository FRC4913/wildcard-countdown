## Getting Started

1. Clone the repo
    ```sh
    git clone https://github.com/FRC4913/wildcard-countdown.git
    ```
2. Install NPM packages
    ```sh
    bun install
    ```
3. Run dev server
    ```sh
    bun run dev
    ```
